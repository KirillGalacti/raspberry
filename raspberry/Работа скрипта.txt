ID нажатой кнопки, на странице выбора лабораторной работы (index.html), 
присваивается кнопке, которая находить на странице выбора обучения или контроля (main.html).
От присвоенного ID зависит какой файл будет открыт по клику кнопки "Обучение".
Так же от присвоенного ID зависит какой тест по пройденному материалу будет открыт по клику кнопки "Контроль"
